Описание web-интерфейса тестового

View Tables
Отображает состояние таблиц БД.
Таблица Lord's - содержит всех повелителей.
Таблица Planet's - содержит все планеты.

Modify Tables
Имеет следующие пункты, согласно условиям тестового:

  • Добавление Повелителя планет в таблицу Lords
  • Добавление Планеты в таблицу Planets
  • Назначить Повелителя управлять Планетой
  • Уничтожить Планету в таблице Planets
    • Проверка результата во вкладке "View Tables".

  • Вывести ТОП-10 самых молодых Правителей
  • Вывести всех Правителей без планет(бездельники)
    • Будет произведено перенаправление на новую страницу с отображением результатов.

About
Описание условий тестового: задание, базовые характеристики сущностей, поддерживаемые методы API.
